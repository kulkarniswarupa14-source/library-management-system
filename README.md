Library management system

1.Project Description
-This is a Java-based application to manage students, courses, and enrollments.

2.Features
- Add Student
- Manage Courses
- Enroll Students
- Input Validation
- Exception Handling

3.Technologies Used
- Java
- OOP (Object-Oriented Programming)

4. Project Structure
- entity → Models (Student, Course, Person)
- service → Business logic
- exception → Custom exceptions
- util → Utility classes
- ui → Main class

5. How to Run
1. Compile:
   javac ui/Main.java

2. Run:
   java ui.Main
